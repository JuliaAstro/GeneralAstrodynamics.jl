    GeneralAstrodynamics.AstrodynamicsCore.Orbit â€” Method

    A simpler constructor for a Synodic CR3BP state with or without the local linearization (state transition matrix) included.

    Please note that this is a convenience constructor for quick calculations. This method is not type stable, since the kwarginclude_stm directly affects the output type of the function. This sub-optimal code is great for quickly constructing an initial state, but this method will be slower than an explicit CircularRestrictedThreeBodyOrbit constructor.
