    CommonTypes

    The following abstract types are defined, which are common parent types for all submodules within Astrodynamics.

    UnitfulAstrodynamics.CommonTypes.AbstractBody — Type

    Abstract type for bodies in space: both CelestialBodys (in TwoBody.jl), and Bodys (in NBody.jl).

    source
    UnitfulAstrodynamics.CommonTypes.OrbitalSystem — Type

    Abstract type describing all states in select Astrodynamics problems.

    source
    UnitfulAstrodynamics.CommonTypes.AbstractTrajectory — Type

    Abstract type describing a collection of states resulting from numerical integration
